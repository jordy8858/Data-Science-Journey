# Data-Science-Journey
Data Science Journey: From Fundamentals to Machine Learning
Welcome to my Data Science portfolio! This repository documents my complete technical progression through the Hyperiondev Data Science Bootcamp.

As a professional in automotive internet sales, I have always relied on understanding customer behavior, performance metrics, and sales trends. I built this repository to showcase my transition into the data science industry, highlighting how I evolved from writing basic Python scripts in PyCharm to developing complex machine learning pipelines and neural networks to solve real business problems.

🚀 The Journey & Progression
This repository is ordered chronologically to demonstrate my learning curve and the expanding complexity of my skillset.

Phase 1: Python Programming Fundamentals
The foundation. Focused on control flow, data structures, and building functional applications.

01_finance_calculators.py: A foundational program utilizing conditional logic to calculate investment returns and home loan repayment schedules.

02_task_manager.py: A comprehensive task delegation system demonstrating file I/O operations, loops, and user authentication.

03_shelf_track.py: An inventory management script utilizing object-oriented programming concepts.

Phase 2: Exploratory Data Analysis (EDA)
Transitioning from software engineering to data analysis, focusing on extracting insights from raw data.

05a_Automobiles.ipynb & 05b_EDA Report - Automobiles.pdf: A deep dive into automotive datasets, utilizing Pandas and Matplotlib to uncover trends, visualize distributions, and present business-facing insights in a formal report.

Phase 3: Predictive Modeling (Supervised Learning)
Applying machine learning engineering concepts to predict continuous outcomes and classify customer behavior.

06_Linear_Regression_Ames.ipynb: Built a regression model to predict housing prices, focusing on feature selection, model training, and performance evaluation.

07_Telco_customer_churn.ipynb: Developed a classification model to predict customer retention, a critical metric for any sales and service-oriented business.

Phase 4: Advanced Machine Learning & Deep Learning
Tackling complex, unstructured data using advanced algorithms.

08_unsupervised_task.ipynb: Applied clustering techniques (like K-Means) to segment data without pre-existing labels.

09_neural_network_task.ipynb: Engineered and trained a neural network, showcasing an understanding of deep learning architectures and epoch-based training.

Phase 5: Natural Language Processing (NLP)
Extracting sentiment and meaning from text-based data.

10a_Sentiment_analysis.ipynb & 10b_Sentiment Analysis Report.pdf: Built an NLP model to classify the polarity of text data, complete with a professional report detailing the model's accuracy and business applications.

🛠️ Tools & Technologies Used
Languages: Python

Environments & Version Control: VSCode, Jupyter Notebooks, Git, GitHub

Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn, NLP Libraries

💡 Key Takeaways
Going through this intensive bootcamp allowed me to marry my hands-on sales experience with advanced data analytics. Whether it is predicting customer churn or forecasting inventory needs, I am passionate about leveraging machine learning to drive ROI, optimize processes, and make data-informed business decisions.
